ART-003: CPE Credit Rules

Type: Rule   Draft   Source: authored   Effective: 2026-02-01

Description

Defines how continuing education credits are calculated, categorized, and validated. Credits are grouped by category with minimum thresholds for renewal. Activities must include audit metadata, provider information, and completion dates.

Anchors

  • Credit categories
  • Minimum thresholds
  • Activity audit metadata

Notes

Draft policy pending final approval from compliance committee.
